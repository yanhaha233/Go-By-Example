### 变量

Go 中的变量需要显示声明，并在函数调用等情况下，编译器会检查其类型的正确性。

```go
package main

import "fmt"

func main3() {
	var a = "initial"      // var 声明一个或多个变量
	fmt.Println(a)

	var b, c int = 1, 2    // 可以一次性声明多个变量
	fmt.Println(b, c)

	var d = true           // Go会自动推断已有初始值的变量的类型
	fmt.Println(d)

	var e int              // 声明后没有给出对应的初始值，变量将会初始化为零值
	fmt.Println(e)         // int零值是0

	f := "short"           // :=语法是声明并初始化变量的简写
						   // var f string = "short"
	fmt.Println(f)
}

```
