### 多返回值

Go 原生支持 _多返回值_。 这个特性在 Go 语言中经常用到，例如用来同时返回一个函数的结果和错误信息。

```go
package main

import "fmt"

func vals()(int,int){
    return 3,7  // (int, int) 在这个函数中标志着这个函数返回 2 个 int
}

func main13(){
    a,b := vals()   // 这里我们通过 多赋值 操作来使用这两个不同的返回值
    fmt.Println(a)
    fmt.Println(b)

    _,c := vals()  // 如果你仅仅需要返回值的一部分的话，你可以使用空白标识符 _。
    fmt.Println(c)
}

```
