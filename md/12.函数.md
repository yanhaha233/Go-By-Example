### 函数

函数是 Go 的核心。

```go
package main

func plus(a int,b int) int{
    return a + b
    // 这里是一个函数，接受两个 int 并且以 int 返回它们的和
    // Go 需要明确的 return，也就是说，它不会自动 return 最后一个表达式的值
}

func plusPlus(a,b,c int) int{
    return a + b + c
    // 当多个连续的参数为同样类型时， 可以仅声明最后一个参数的类型，忽略之前相同类型参数的类型声明。
}

func main12(){
    res := plus(1,2)
    fmt.Println("1+2=",res)

    res = plusPlus(1,2,3)
    fmt.Println("1+2+3=",res)
    // 通过 函数名(参数列表) 来调用函数
}
```
