### 值

Go 拥有多种值类型，包括字符串、整型、浮点型、布尔型等。

```go
package main

import "fmt"

func main2() {
	fmt.Println("go" + "lang")      // 字符串可以通过 + 连接
	fmt.Println("1+1 = ", 1+1)      // 整数
	fmt.Println("7.0/3.0 =", 7.0/3.0) // 浮点数
	fmt.Println(true && false)  //常见的布尔型操作
	fmt.Println(true || false)
	fmt.Println(!true)
}

```
